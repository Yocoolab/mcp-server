# Security Policy

## Reporting a vulnerability

If you discover a security issue in `@yocoolab/mcp-server`, **please do not open a public GitHub issue**. Instead, email **support@attune-dating.com** with:

- A description of the vulnerability
- Steps to reproduce, or a proof-of-concept
- The version of the package affected
- Your name / handle for credit (optional)

We aim to acknowledge reports within 2 business days and to ship a fix or mitigation within 14 days for high-severity issues.

## Supported versions

We provide security fixes for the latest minor release of the current major version. Older majors are not supported once a new major has been released for 90 days.

| Version | Supported |
|---------|-----------|
| 1.x     | ✅        |

## Scope

In scope:
- Code in this repository
- Published `@yocoolab/mcp-server` npm package

Out of scope (please report to Yocoolab via the relevant channel instead):
- The Yocoolab API at `app.yocoolab.com`
- The Yocoolab Chrome extension
- The Yocoolab web dashboard
